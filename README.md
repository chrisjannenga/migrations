# Migration Commands
---
## Migrate

1) `knex init`
2) `mkdir db`
3) `touch db/knex.js`
4)





### To create a migration file use :

`knex migrate:make users`

### To Roll over to the most current migration use :

`knex migrate:latest`

### To Rollback to the previous migration use :

`knex migrate:rollback`
